# Homebrew tap
Homebrew tap repository that provides formula for public Edgee binary, [Edgee CLI](https://github.com/edgee-ai/edgee).

## Install

To install `edgee` formula:

```
brew install edgee-ai/tap/edgee
```

## Issues
If you encounter any non-security-related bug or unexpected behavior, please file an issue on the corresponding project repository (i.e. [edgee](https://github.com/edgee-ai/edgee/issues), not this one.


### Security issues
Please see our [SECURITY.md](SECURITY.md) for guidance on reporting security-related issues.
